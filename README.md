# CUIMS_AUTOLOGIN_EXTENSION
I developed CU Auto Login, a 𝐂𝐡𝐫𝐨𝐦𝐞 𝐄𝐱𝐭𝐞𝐧𝐬𝐢𝐨𝐧 that automates the login process for 𝐂𝐡𝐚𝐧𝐝𝐢𝐠𝐚𝐫𝐡 𝐔𝐧𝐢𝐯𝐞𝐫𝐬𝐢𝐭𝐲 portals: CUIMS and CU LMS.  Manual credential entry is a repetitive task for students. This tool removes that friction while keeping user data secure on the local machine.
